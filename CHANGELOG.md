#### [0.5.0](https://github.com/cgaeking/n8n-nodes-enlyst/compare/0.4.14...0.5.0)

> 8 November 2025

**Breaking Changes:**
- Removed: Referral resource and "Get Stats" operation (referral statistics feature removed)

**New Features:**
- Added: Find Leads operation - search for leads via Google Maps
  - Optional project assignment with toggle
  - Location search by name OR coordinates
  - Automatic geocoding via OpenStreetMap Nominatim API
  - Preserves all external data from search results
- Added: Add Leads operation - import leads from external sources with custom data

**Improvements:**
- All external data from lead searches (Google Maps, etc.) now preserved in `externalData` field
- Changed field prefix from `serper_` to `googleMaps_` for clarity
- Nested objects automatically JSON stringified for CSV export compatibility
- External data displayed in expandable rows with tabbed interface in frontend
